# CLAUDE.md — page（官网 ezagent.cloud）

EZAgent 官方网站。License: Apache 2.0。

## 定位

- 产品介绍与品牌展示
- 文档入口导航
- 下载与快速开始指引

## 开发指南

### 包管理

- 使用 `pnpm`（禁止 `npm` / `npx`）
- 安装依赖：`pnpm install`
- 开发服务器：`pnpm run dev`
- 构建：`pnpm run build`

### 品牌素材

- Logo 和样式素材位于 `docs/style/`
- 使用项目统一的视觉风格

### 内容规范

- 产品描述与 `docs/README.md` 保持一致
- 技术术语保留英文原文
- 支持中英文双语展示

## Commit scope

```
feat(page): add hero section
fix(page): fix responsive layout on mobile
```
