## 选择你的方式

EZAgent 提供多种使用方式，选择最适合你的：

### 桌面应用

完整的图形界面体验。内置 Python 引擎 + FastAPI + React WebView。

- **macOS**: DMG 安装包
- **Windows**: MSI 安装程序
- **Linux**: AppImage

*即将推出 — 加入等候名单获取通知*

### 命令行工具

面向开发者的命令行界面。通过 pip 安装，一条命令启动。

```bash
pip install ezagent
ezagent start
```

浏览器自动打开 Web 界面。

### Homebrew (macOS)

```bash
brew install ezagent
```

## 系统要求

- Python 3.10+
- 4GB+ RAM 推荐
- 支持 macOS、Windows、Linux
